# Beyond-the-Beat
Compared audio-only, lyrics-only, and multimodal models for music genre classification using SAS Viya and Python, showing that multimodal models outperform single-source approaches. Found that explicit content is poorly predicted from audio features alone, highlighting the importance of lyrical context. Written for the SAS Curiosity Cup 2026.

Original Dataset: https://www.kaggle.com/datasets/serkantysz/550k-spotify-songs-audio-lyrics-and-genres
